# ETF 定期定額試算網站需求

## 網站目標
建立一個 ETF 定期定額試算網站，讓使用者可以輸入每月投入金額、投資年限、預估年化報酬率，計算長期累積成果。

## 主要功能
1. 輸入每月投入金額
2. 輸入投資年限
3. 輸入預估年化報酬率
4. 顯示總投入金額
5. 顯示預估累積金額
6. 顯示投資收益
7. 顯示每年累積資產變化

## 介面需求
1. 使用繁體中文
2. 手機與電腦都能正常瀏覽
3. 操作簡單，適合一般使用者
4. 畫面風格乾淨、清楚
5. 不需要登入

## 技術需求
1. 使用 HTML、CSS、JavaScript
2. 不使用後端資料庫
3. 可部署到 GitHub Pages 或 Cloudflare Pages
4. 所有計算在瀏覽器端完成

## 後續可能新增功能

2. 加入匯率換算
3. 加入每月投入日期設定
4. 加入下跌加碼模擬
